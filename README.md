# Dashboard
Untuk instalasi silahkan jalankan command dibawah ini

1. `git clone https://github.com/vermaysha/Dashboard.git`
2. `cd Dashboard`
3. `npm install`
4. `composer install`
5. `php -r "file_exists('.env') || copy('.env.example', '.env');"`
6. `php artisan key:generate --ansi`
7. `php artisan serve`

Selesai
